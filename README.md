As a developer, I believe in working smart and building on proven solutions. Using an open-source template allowed me to:

- Spend time on content rather than reinventing the wheel

## Setup & Installation

```bash
# Clone the repository
git clone https://github.com/Priya-995/my-portfolio.git

# Navigate to project directory
cd my-portfolio

# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build
```
